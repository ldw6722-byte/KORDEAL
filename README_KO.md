한국 제조사·공방·장인과 전세계 바이어를 하나의 플랫폼에서 연결합니다.

---

## KORDEAL이란?

KORDEAL은 **한국 공급사**와 **해외 바이어**를 연결하는 B2B 도매 소싱 플랫폼입니다.
중국의 1688.com처럼, 전세계 바이어가 한국 상품을 쉽고 안전하게 소싱할 수 있는 창구를 만듭니다.

> *"한국 상품 소싱, 이제 이렇게 하세요."*

---

## 왜 KORDEAL인가?

- 🌍 **글로벌 퍼스트** — 영문·다국어 지원, 글로벌 결제 내장
- 🔒 **에스크로 결제** — 바이어·공급사 양방향 보호, 먹튀 방지
- 📋 **무역 규제 안내** — 국가 × 카테고리별 수출입 규제 사전 안내
- 🏺 **장인마켓** — 한국 공방·장인의 희소성 수작업 상품 전문 카테고리
- ⚡ **AI 자동 번역** — 공급사는 한국어로 등록, 영문 페이지 자동 생성

## 장인마켓 (Jangin Market)

**한국 공방·장인이 만드는 희소성 상품** 전문 카테고리입니다.
도예·금속공예·은공예·나전칠기·전통 자수 등, 세계 어디에도 없는 한국 고유의 수작업 제품을 전세계 전문 바이어와 연결합니다.

양산품이 아닌, 스토리가 있는 상품. 셀러와 바이어가 함께 성장하는 구조입니다.

---

## 에스크로 결제 흐름

```
바이어 결제 → 에스크로 보관 → 공급사 발송
→ 바이어 수령 확인 → 공급사 정산
```

분쟁 발생 시 72시간 내 중재 판정.
