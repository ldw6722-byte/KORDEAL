| 날짜 | 태스크 | 산출물 |

| 7/25 (토) | 주문 UI + Stripe Elements 결제창 붙이기 (결제 로직 제외, UI만) | 주문 화면 |
| 7/26 (일) | 리뷰 + 버퍼 | — |

| 7/29 (수) | 전체 UI 플로우 연결 + RLS 권한 테스트 | 통합 동작 확인 |
| 7/30 (목) | 최종 QA, 버그 수정, 화면 정리 | ✅ Phase 2 MVP 완료 |

### Phase 2에서 Supabase로 처리 가능한 항목

- 회원가입 / 로그인 (Auth SDK)
- 상품 등록 / 조회 (PostgreSQL + RLS)
- 채팅 메시지 실시간 처리 (Realtime)
- 파일 업로드 — 사업자등록증 PDF, 상품 이미지 (Storage)

### Phase 2에서 Spring Boot 없이 처리 불가한 항목 (Phase 3으로 이관)

- 에스크로 정산 엔진 (상태 전이 자동화)
- 환율 계산 모듈
- Claude API 번역 파이프라인
- 무역 규제 매트릭스 엔진 (FDA, CE 등)
- 실제 결제 처리 (Stripe / 토스 / PayPal 연동)
